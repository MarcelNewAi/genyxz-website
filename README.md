# website-template-system

## Purpose

This app is used to generate complete website blueprint documents from a single guided form.

It produces:
- `MASTER_PLAN.md`
- `docs/*.md` implementation guides
- `CODEX_PROMPTS.md` copy-paste execution prompts

Use it to standardize project setup, reduce manual planning time, and keep delivery steps consistent.

> All build instructions are in [`MASTER_PLAN.md`](./MASTER_PLAN.md)

## Company Context

Projekt deluje v okolju preventive, sodobne medicine in dolgoročne zaščite, povezan z blagovno znamko Dr Best / Trilogija zaščite. Cilj je zgraditi občutek resnosti, premium brand prisotnost in usmeriti obiskovalce na rekruting landing page za prijavo. Stran mora odgovoriti na vprašanje 'Kaj je GenYXZ?' in zgraditi občutek 'To je nekaj resnega in zanimivega'.
